Detailed Description

template<typename Scalar>
class ChefDevr::BRDFReconstructor< Scalar >

Class that allows BRDF reconstruction from latent space coordinates.

Template Parameters
ScalarThe type of the values used to reconstruct a BRDF. The precision of this type is crucial to reconstruct an accurate BRDF from the latent space.
